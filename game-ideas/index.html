Find it!

In this game the goal is to find random objects spred throughout your home.

  • Make a card for as many random items as you would like. This number really depends on how many player you have
  • Click the "Pick One" button
  • All the players then race to find the object picked!
  • The player that finds the object first gets a point
  • The first player to get 10 points wins!
